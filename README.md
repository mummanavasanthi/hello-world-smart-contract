# HelloWorld Smart Contract

This project demonstrates the deployment of a simple HelloWorld smart contract on the SecureChain (SCAI) Mainnet using Solidity, Remix IDE, and MetaMask wallet.

## Technologies Used

- Solidity
- Remix IDE
- MetaMask
- SecureChain Mainnet (SCAI)

## Deployment Process

1. Installed MetaMask wallet
2. Added SecureChain Mainnet network
3. Received SCAI faucet tokens
4. Created smart contract in Remix IDE
5. Compiled Solidity contract
6. Connected MetaMask with Remix
7. Deployed contract on SecureChain Mainnet

## Tools & Platforms

- Remix IDE
- MetaMask Wallet
- SecureChain Faucet

## Outcome

Successfully deployed the HelloWorld smart contract on SecureChain Mainnet.
